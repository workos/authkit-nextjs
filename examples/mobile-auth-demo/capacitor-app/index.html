This demo shows how OAuth works in a native mobile app using:

  • Custom URL Scheme: workosauthdemo://
  • Browser Plugin: Opens system browser
  • App Plugin: Captures the callback
  • Backend: Exchanges code for tokens

What happens when you click:

  1. App opens system browser with WorkOS URL
  2. You authenticate in WorkOS
  3. WorkOS redirects to: workosauthdemo://callback?code=...
  4. iOS/Android captures this and returns to app
  5. App extracts code and calls backend
  6. Backend exchanges code for tokens
